وسائط التخزين الأولية Primary Storage  :-
وهي تشمل وسائط التخزين التي تتعامل مباشرة مع المعالج مثل الذاكرة الرئيسية Main Memory  والــ  Cache Memory  وهذه الوسائط تمتاز بسرعتها العالية ولكن حجمها صغير مقارنة بالذاكرة الثانوية .
